Research Knowledge Base

Build a NotebookLM knowledge base over your own sources that answers questions with citations — real retrieval (RAG), not the model guessing. Each sample below is a worked reference: a source list, five cited Q&A, and a one-page briefing.

Subscription-first · no API key · no code. Built entirely in Google NotebookLM (free with a Google account). Add real sources → ask questions that return cited answers → generate a briefing and an Audio Overview. NotebookLM is RAG — retrieval + generation over your material. That's the honest step up from HO4's in-context grounding.
