Research Knowledge Base

Your starting point. Pick ONE template below, then build a NotebookLM knowledge base over your own sources that answers questions with citations — real retrieval (RAG). No API key, no code.

This is the starter branch. Each folder is a copy-and-use template — a sources.md, qa.md and briefing.md with example content filled in and << … >> blanks for your own material. Build the notebook in NotebookLM, then fill the blanks. For a complete reference, switch to the main branch.
