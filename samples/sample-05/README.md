# HO5 Sample 5 — Finance Calculator

## Your task

People weighing a loan or savings goal cannot picture how the numbers play out. Build an interactive calculator Artifact that shows compound growth or loan repayments on a chart.

## What to do

1. Open index.html in your browser — you should see the skeleton layout (requires internet for Chart.js CDN)
2. Read the TODO comments in the HTML and JavaScript
3. Fill in: the input labels and defaults, the month-by-month calculation logic, the summary stats, and the chart data
4. Test it with a variety of realistic numbers
5. Push to GitHub and enable Pages for a shareable link

## Inputs your tool needs

- Principal amount
- Interest rate (%)
- Term (months or years)
- Optional: monthly contribution

## Outputs your tool should show

- Final value or total repaid
- Month-by-month chart (Chart.js)
- Key stats: total interest, break-even point
